Maker Faire 2025 – Pitching Machine

For Maker Faire 2025, I designed and built a pitching machine capable of consistently launching baseballs using a controlled mechanical system. This project required planning, budgeting, testing, and iteration.

Project Summary

The pitching machine was built using readily available materials and focused on reliability, safety, and repeatability. The final design successfully demonstrated mechanical principles and problem-solving skills.

Design & Challenges

Throughout the build process, challenges included alignment, power transmission, and consistency. Each issue was addressed through testing and redesign.

Documentation

Budget spreadsheets, daily logs, and build documentation were kept throughout the project to track progress and decisions.
